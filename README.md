# Sistema Web Gestión Comercial para Vivero

## Ejecutar proyecto

- Crear una base de datos PostgreSQL con el nombre `swgcv`

- Crear una copia del archivo `.env.example` con el nombre `.env` y configurarlo

- Instala las dependencias con `npm install`

- Ejecuta el seed con `npm run seed`

- Iniciar el servidor de desarrollo con `npm run start:dev`
